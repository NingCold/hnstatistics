# 频率统计分析软件

一个用于对文本或数据进行频率统计与分析的桌面应用程序，提供项目化管理、分析结果对比以及灵活的提交模式。

---

## 界面说明

### 左侧：项目栏
- 顶部提供搜索框，可用于快速筛选项目
- 单击选中项目
- 双击打开项目

### 右侧：分析区域
- 上方为 **输入栏**，用于输入待分析内容
- 下方为 **结果栏**，用于展示分析结果

---

## 使用流程

1. **打开项目**
   - 双击左侧项目列表中的项目
   - 打开后，结果栏会显示该项目当前已有的分析结果

2. **进行分析**
   - 在输入栏中输入或粘贴需要分析的内容
   - 点击 **Analyze** 按钮开始分析

3. **切换提交模式（Commit Mode）**
   - 点击 **Commit Mode** 按钮可切换提交策略：
     - **Overwrite**
       - 使用输入栏的分析结果直接覆盖当前项目结果
     - **Merge**
       - 将输入栏的分析结果与当前项目的结果进行合并

4. **提交结果**
   - 点击 **Commit** 按钮提交分析结果
   - 提交方式由当前选中的 Commit Mode 决定
